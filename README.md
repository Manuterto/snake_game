# snake_game

--Snake Game🐍--

A simple Snake game made with Python and Pygame.

# -Features-

- Snake movement
- Food system
- Score counter
- Collision detection
- Game over screen

# -Technologies-

- Python
- Pygame

-Installation-

Install pygame:

```bash
pip install pygame
```

Run the game:

```bash
python snake.py
```

-Controls-

- Arrow keys to move
